# letterboxd-picker

Start a webserever to pick a random film from a letterboxd watchlist or public list 

## Usage
- Clone repo
    ```
    git clone https://github.com/lennysgarage/letterboxd-picker.git
    ```
- Get packages
    ```
    cd letterboxd-picker
    go get
    ```
- Run server tool
    ```
    cd letterboxd-picker
    go build
    ./letterboxd-picker
    ```

## Example
    ./letterboxd-picker
